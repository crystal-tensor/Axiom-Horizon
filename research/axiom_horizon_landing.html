B1/B7 / merged

gcm_h6 从 w8_21 数值搜索转入 symbolic scaffold

T-B7-009 与 T-B7-010a 已把 local-search 空间压到明确边界：43,480 次 w8_21 optimizer runs 后仍无 exact occurrence-removing certificate。

Evidence

12 个 retained templates 中 0 个 single-template one-angle route 清除 1.20x STV 边界；最佳 w8_21 仍至少需要每 occurrence 移除 2 个 arbitrary rotations。

Next Gate

停止继续堆同类 numerical search；下一步只接受 symbolic KAK/Clifford scaffold 或不同 occurrence-removing rewrite。

B2 / negative boundary

leakage-aware decoder 路线已进入硬件式观测模型压力测试

最新 hardware-like leakage model gate 消耗 detector bitstrings，不再只依赖 synthetic flag fixture，但仍不是 calibrated hardware 或 production decoder。

Evidence

3 个 strict challenge rows、3 个 observation profiles、1,728 profile-shots 与 864 holdout shots；best conservative profile 415 flag events，0 prediction changes，0 fixed failures，22 injected failures 保持不变。

Next Gate

需要真实/校准 leakage traces、shot-conditioned decoder contract，或证明当前 DEM-edge posterior 注入机制为什么应当改变 PyMatching 决策。

B10/B8 / verifier bridge

量子优势验证从 toy leakage 进入 noisy circuit bridge

B10-T2 已完成 Qiskit/Aer circuit-level、noisy Aer 和 backend-calibrated-style 三层桥接；refresh mode 仍是必要防线。

Evidence

9,600 noisy Aer parity-verifier circuits：safe refresh modes 下 honest acceptance 1.0、adversary acceptance 0.0；GenericBackendV2-style bridge 中 adversary acceptance 升到 0.25，暴露校准敏感性。

Next Gate

把 GenericBackendV2 快照替换成真实 backend properties，或直接做 hardware randomized-measurement verifier execution。

B5/B10 / same-access guardrail

response-sampler cost stress 继续拒绝轻率量子优势叙事

同 9 个 D5 Hubbard response rows 上，乐观 bounded-density finite-difference sampler 仍无法追平 exact-state-seeded MPS pressure。

Evidence

匹配 seeded-MPS pressure 的 shot lower bound 为 min/median/max 3.861e9 / 7.645e12 / 2.849e29；0 rows by shots beat explicit D5 matvec ops。

Next Gate

要么补 mature canonical-environment production DMRG/MPS，要么给出真实 same-access response oracle 的 state-preparation、mixing、measurement 与 confidence 成本。
